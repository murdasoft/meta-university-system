## 💡 Опционально: Показывать какой движок использовался

Если хотите показать пользователю "Распознано через ChatGPT":

### Можно добавить в ответ API (попросите бэкенд):
```json
{
  "text": "Распознанный текст",
  "confidence": 0.95,
  "engine": "chatgpt-4-vision"  // ← новое поле
}
```

### Тогда в Flutter можно показать:
```dart
if (response['engine'] == 'chatgpt-4-vision') {
  // опционально: пометка «точнее за счёт OpenAI», если engine = chatgpt-4-vision
}
```

Но это **опционально**! Можно вообще не показывать.
