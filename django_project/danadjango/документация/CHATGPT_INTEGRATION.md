# 🤖 Интеграция ChatGPT-4 Vision для распознавания текста

## Поведение на бэкенде

### 1. Добавлена поддержка OpenAI API

**Гибридный подход распознавания:**
```
1. Google Vision API (быстро, дёшево) → если confidence >= 80%
   ↓
2. ChatGPT-4 Vision (точнее, дороже) → если Google не уверен
   ↓
3. Tesseract OCR (бесплатно) → fallback если оба не сработали
```

### 2. Обновлена модель APIKey

Поддерживаются три типа ключей:
- `google_vision` - Google Vision API
- `openai` - OpenAI (ChatGPT-4 Vision)
- `other` - Другие API

### 3. Настройки в админ-панели

Администратор может добавить OpenAI API ключ:
1. Зайти в http://109.248.32.73/dashboard/settings/
2. Нажать "Добавить ключ"
3. Выбрать тип: **OpenAI (ChatGPT-4 Vision)**
4. Вставить API ключ (начинается с `sk-...`)
5. Сохранить

### Сценарий 1: Хорошее качество фото
```
Пользователь → Фото → Google Vision (0.5 сек) → ✅ Confidence 95%
                                                 ↓
                                            Результат
```
**Затраты:** $0.0015 за запрос

### Сценарий 2: Плохое качество / рукописный текст
```
Пользователь → Фото → Google Vision (0.5 сек) → ⚠️ Confidence 65%
                                                 ↓
                                    ChatGPT-4 Vision (2 сек) → ✅ Исправил текст
                                                 ↓
                                            Результат
```
**Затраты:** $0.0015 + $0.01 = $0.0115 за запрос

### Сценарий 3: Оба API недоступны
```
Пользователь → Фото → Google Vision → ❌ Ошибка
                                      ↓
                          ChatGPT-4 Vision → ❌ Ошибка
                                      ↓
                              Tesseract OCR → ✅ Базовое распознавание
                                      ↓
                                  Результат
```
**Затраты:** $0 (Tesseract бесплатный)

## ❓ FAQ

**Q: Нужно ли обновлять Flutter приложение?**
A: Нет! API не изменился, всё работает как раньше.

**Q: Как отключить ChatGPT если он дорогой?**
A: Удалить OpenAI API ключ из настроек. Бэкенд вернётся к Google Vision + Tesseract.

**Q: Можно ли использовать только ChatGPT?**
A: Да, но это будет дороже. Нужно изменить логику в `ocr/services.py`.

**Q: Работает ли это прямо сейчас?**
A: Да, после добавления OpenAI API ключа в настройках.

## Кратко

- В цепочке OCR при низкой уверенности Google Vision может подключаться OpenAI (тип ключа `openai` в настройках).
- Формат ответа API для клиента можно не менять; при необходимости в теле ответа указывается поле `engine`.
- Чтобы отключить дорогой путь: убрать OpenAI API ключ из настроек — останутся Google Vision и Tesseract по текущей логике в `ocr/services.py`.
