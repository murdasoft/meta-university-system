## 🔒 Безопасность

### Активные порты:
- **22** (SSH) - для удаленного доступа
- **80** (HTTP) - для веб-приложения
- **443** (HTTPS) - зарезервирован для SSL

### Firewall (UFW):
```bash
# Проверка статуса
sudo ufw status

# Открытые порты
22/tcp    ALLOW       Anywhere
80/tcp    ALLOW       Anywhere
443/tcp   ALLOW       Anywhere
```
