## 📦 Основные компоненты

### 1. Django Приложения

#### accounts (Пользователи)
- **Назначение**: Управление пользователями и аутентификация
- **Модели**: CustomUser (расширенная модель пользователя)
- **API**: CRUD операции с пользователями, JWT токены
- **Веб**: Страницы входа/выхода

#### ocr (Распознавание текста)
- **Назначение**: OCR функциональность
- **Модели**: Language, OCRRequest, OCRSettings
- **API**: Загрузка изображений, получение результатов
- **Сервисы**: OCRService для обработки изображений

#### dashboard (Дашборд)
- **Назначение**: Админ панель и статистика
- **Модели**: DashboardStats
- **API**: Статистика системы
- **Веб**: Красивый интерфейс с графиками

### 2. Конфигурационные файлы

#### settings.py
```python
# Основные настройки:
- SECRET_KEY: ключ шифрования
- DEBUG: режим отладки (False в продакшене)
- ALLOWED_HOSTS: разрешенные хосты
- DATABASES: подключение к PostgreSQL
- INSTALLED_APPS: установленные приложения
- MIDDLEWARE: промежуточное ПО
- TEMPLATES: настройки шаблонов
- STATIC_URL/ROOT: статические файлы
- MEDIA_URL/ROOT: медиа файлы
```

#### urls.py
```python
# Маршруты:
/                         → редирект на /login/
/login/                   → страница входа
/logout/                  → выход
/dashboard/               → дашборд
/admin/                   → Django Admin
/api/accounts/            → API пользователей
/api/ocr/                 → API OCR
/api/dashboard/           → API статистики
/swagger/                 → API документация
```

### 3. Системные файлы

#### /etc/systemd/system/qazaqdana.service
```ini
[Unit]
Description=Qazaq Dana Django Application

[Service]
Type=notify
User=qazaqdana
Group=www-data
WorkingDirectory=/var/www/qazaqdana
ExecStart=/var/www/qazaqdana/venv/bin/gunicorn danadjango.wsgi:application

[Install]
WantedBy=multi-user.target
```

#### /etc/nginx/sites-available/qazaqdana
```nginx
server {
    listen 80;
    server_name 109.248.32.73;
    
    location /static/ {
        alias /var/www/qazaqdana/staticfiles/;
    }
    
    location /media/ {
        alias /var/www/qazaqdana/media/;
    }
    
    location / {
        proxy_pass http://unix:/var/www/qazaqdana/gunicorn.sock;
    }
}
```

## 🔐 Файлы с чувствительными данными

### Не коммитить в Git:
```
.env                    # Переменные окружения
*.pyc                   # Скомпилированный Python
__pycache__/            # Кэш Python
venv/                   # Виртуальное окружение
db.sqlite3              # Локальная БД SQLite
media/                  # Загруженные файлы
*.log                   # Логи
.DS_Store              # macOS файлы
```

### Безопасное хранение:
- **.env** файлы хранить локально
- **Пароли** в менеджере паролей
- **SSH ключи** в ~/.ssh/
- **Бэкапы БД** в безопасном месте
